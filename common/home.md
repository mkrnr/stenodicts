# Home 

## Open Applications

```yaml
P-RPL: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt \"PyCharm CE\" PyCharm}' # Open PyCharm
T-PBLG: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt \"IntelliJ IDEA CE\" \"IntelliJ IDEA\"}' # Open IntelliJ
W-B: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt Safari Safari}' # Open Browser
PH-L: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt Mail Mail}' # Open Mail
PW-RPB: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt Brain.fm Brain.fm}' # Open brain.fm (Safari tab)
SKWR-R: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt Jira Jira}' # Open Jira
KH-B: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt ChatGPT ChatGPT}' # Open chat bot
```


## Commands

```yaml
TW-B: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt Safari Safari}{#space}{#command(tab)}' # Toggle play pause in browser
W-FRB: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt Safari Safari}{#left}{#command(tab)}' # Press left in browser
W-BLG: '{PLOVER:SHELL:osascript ~/git/stenodicts/scripts/open-app.scpt Safari Safari}{#right}{#command(tab)}' # Press right in browser
TKHR-PB: '{PLOVER:SHELL:shortcuts run Desk-Lights-On}'
TKHR-F: '{PLOVER:SHELL:shortcuts run Desk-Lights-Off}'
```
