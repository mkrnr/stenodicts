## Special Keys

```yaml
-FRPB: '{}{#backspace}' # Backspace
TK-L: '{}{#delete}' # Delete
H-F: '{}{#escape}' # Escape
TPH-RT: '{}{#insert}' # Insert
R-R: '{}{#return}' # Return
T-B: '{}{#tab}' # Tab
ST-B: '{}{#shift(tab)}' # Shift-Tab
```
